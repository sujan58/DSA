<div class="header"><div class="title">G. Mixing MEXes</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You are given $$$n$$$ <span class="tex-font-style-bf">arrays</span> $$$a_1, a_2, \ldots, a_n$$$. </p><p>The following operation is performed exactly once: </p><ul> <li> Choose any array among $$$a_1, a_2, \ldots, a_n$$$. Suppose you've chosen array $$$a_i$$$ ($$$1 \leq i \leq n$$$). </li><li> Choose any element in array $$$a_i$$$. Suppose you've chosen the $$$j$$$-th element of $$$a_i$$$, denoted by $$$a_{i,j}$$$ ($$$1 \leq j \leq |a_i|$$$, where $$$|a_i|$$$ denotes the length of array $$$a_i$$$). </li><li> Choose any other array among $$$a_1, a_2, \ldots a_n$$$ <span class="tex-font-style-bf">that is not $$$a_i$$$</span>. Suppose you've chosen $$$a_k$$$ ($$$1 \leq k \leq n, k \neq i$$$). </li><li> Add $$$a_{i,j}$$$ to the back of array $$$a_k$$$. Then, remove $$$a_{i,j}$$$ from $$$a_i$$$. </li><li> The value of this operation $$$(i,j,k)$$$ is defined as the sum of each array's $$$\operatorname{MEX}$$$ after the operation is performed. More formally, the value of an operation after the operation is performed is $$$\sum_{i=1} ^{n} \operatorname{MEX}(a_i)$$$. </li></ul><p>Evaluate the sum of the values of all possible distinct independent operations. Two operations are distinct if the ordered triple of integers $$$(i,j,k)$$$ is different.</p><p>$$$\operatorname{MEX}(a)$$$ is defined as the smallest non-negative integer that is not present in the array. For example, $$$\operatorname{MEX}([1, 2, 0, 5])$$$ is $$$3$$$, and $$$\operatorname{MEX}([1, 2, 4, 9])$$$ is $$$0$$$.</p></div><div class="input-specification"><div then contain $$$l_i$$$ integers $$$a_1, a_2, \ldots, a_{l_i}$$$ ($$$0 \le a_{i_j} \le 10^6$$$) — the array $$$a_i$$$.</p><p>It is guaranteed that the sum of $$$l_i$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output the sum of the values of all possible distinct
